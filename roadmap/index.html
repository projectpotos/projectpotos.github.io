# Roadmap

Potos is just in a relatively early phase. We are currently considering:

  • Generic iso build container for Ubuntu 22.04 LTS
  • Basic Ansible-Pull mechanism
  • First Boot Dialogs
  • Integration of specs repository: Overwrite Potos role values and run your own, additional roles in combination with Potos Ansible-Pull.
  • Backport Support for Ubuntu 20.04 LTSopen in new window
  • Generic sudo role
  • Generic firewall role
  • Generic package install role
  • SSSD with ADopen in new window / UCSopen in new window / OpenLdapopen in new window / FreeIPA/RHEL IdMopen in new window
  • Generic AppArmor / SELinux role
  • Generic dconf / gsettings role
  • Printer configuration role
  • OsQueryopen in new window / Wazuhopen in new window role
  • Microsoft Defender ATPopen in new window role
  • Generic Application configuration role
  • Generic Window Manager support
  • Certmonger role for automatic cert renewal
  • LUKS Support for Yubikey
  • Support for "el / dnf" distros
  • Support Ubuntu 24.04 LTS
